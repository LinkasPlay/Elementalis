About the server

Elementalis is hosted on an external server by JOJO, accordingly there may be lag and ocasional crashes, in addition to the daily reboot at 2am UTC. This may be a bit janky, however we did this in order to have full control on the infrastructure, being able to easily update the server to new versions of minecraft or new versions of the mod/plugin.


How to join

I suggest you to follow this very good guide on how to join .

If you are too lazy to read it, here's a summary :

If you're on PC or Mobile :
You just need to add an external server with the IP found on the discord, you don't need to change the port.

If you're on console :
1) Change your DNS Setting from Automatic to Manual (if it were already on Manual take note of the Primary and Secondary DNS if you need to use your old settings)
2) Set the Primay DNS to 104.238.130.180 and the Secondary DNS to 0001.0001.0001.0001
3) Join any Minecraft server and you should be in an empty world with a server list.
4) Add an external server with the IP found on the discord and don't change the port.

You can now play on the Elementalis Server !

If you have issues connecting to the server, contact one of the moderators on discord.
