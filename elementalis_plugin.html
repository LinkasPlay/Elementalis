About the plugin

Elementalis use a custom made plugin with multiple aspects to it.

Economy

The plugin uses a custom economy system where you need to buy access to the shop with 1 Star Point.
The economy is based around the currency of the Omega (Î©).
You can buy or sell a number of different items, the complete list is accessible via the commands /shop buy or /shop sell.
Commands :

/shop buy - Opens the buy shop.
/shop sell - Opens the sell shop.
/economy leaderboards - View the 10 richest players on the server.
/economy bank - See how much money you own.
/economy pay [player] [amount] - Give money to someone else.

Factions

Every faction server needs a way to create, manage and join factions.
Creating a faction will cost you 1 Star Point.
On the server factions are limited to only 20 players maximum and every player in the faction has a role.
Roles :
Player - cannot do anything specific.
Admin - can accept players in the faction, rename the faction and claim chunks.
Owner - can do everything an admin can do but can change players' roles.

Commands :
/f create [name] - Creates a faction with the name you gave.
/f list - Opens a list of every factions on the server.
/f join [name] - Join a faction, if the faction is in "Ask Only Mode" a request will be made.
/f request - See requests to join the faction. (Admins and Owner only)
/f accept [player] - See requests to join the faction. (Admins and Owner only)
/f decline [player] - See requests to join the faction. (Admins and Owner only)
/f leave - Leave you faction .(Will put you in timeout)
/f kick [player] - Kicks a player from the faction. (Will put the player in timeout/Owner only)
/f settings - Opens a menu to change the settings of the faction. (Admins and Owner only)
/f claim - Claims the chunk you are in. (Costs power/Admins and Owner only)
/f unclaim - Unclaims the chunk you are in. (Gives power/Admins and Owner only)

Arena

The server has a custom Arena with it's own leaderboard.
Commands :
/arena invite [name] - Invites a player to a duel.
/arena accept (name) - Accepts an invite from a player or the latest invite you received.
/arena decline (name) - Declines an invite from a player or the latest invite you received.
/arena invitelist - Shows you who invited you.
/arena queue - Shows you the current queue of the arena.

Warps

The plugin allows you to set custom warps to teleport back to later.
Every player has 3 warps available at the start but they can buy more (up to 4 warps) with Star Points.

Commands :
/warps list - Lists all you warps.
/warps set [name] - Creates a new warp or move the warp with the same name.
/warps remove [name] - Removes a warp.
/warps tp [name] - Teleports you to the warp.
/warps reset - Removes every warps you have.
